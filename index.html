O que é o HTML?

O HTML (Hyper Text Markup Language, ou linguagem de marcação de textos), é uma linguagem de marcação e não uma linguagem de programação, como muitos pensam. Ele é o componente responsável pela estrutura, definição do posicionamento dos elementos, estruturar a divisão de uma página em blocos visuais e combinar texto, imagens e vídeos de uma aplicação web.

Foi criada em Dezembro de 1990, por Tim Berners-Lee, britânico que também criou a World-Wide-Web (www).

Elementos

Elemento em HTML representa um item presente na aplicação web, como por exemplo um parágrafo imagens, links, e outros tantos recursos.

Tags

As tags servem para denominar um elemento. Ela vai determinar a função que este elemento terá dentro da página web. Elas devem ser escritas entre os símbolos "<" e ">". Podem ser utilizadas sozinhas, ou em conjunto com a tag de fechamento, onde há adição do caractere "/".

Veja os exemplos para melhor entendimento.

Exemplos

Utilizaremos como exemplo a tag "p", que nos possibilita escrever um parágrafo em nossa aplicação.

<p>Este é um parágrafo</p>

Perceba que o caractere concebido à tag do parágrafo, o "p", se inicia entre os símbolos "<" e ">", e após a finalização do parágrafo, usamos além dos dois símbolos iniciais, o símbolo "/" para sinalizar que é uma tag de fechamento, e que ali se dá o fim do parágrafo.

Atributos

Os atributos nos permite adicionar características e funcionalidades para elementos.

Importante destacar que existem os atributos globais, que funcionam para todos os elementos, e os atributos específicos, que funcionam para um elemento em especial.
